# Sprites

A collection of ready-to-use visual novel character sprites tailored for [Ren'Py](https://www.renpy.org/).

Also includes a custom screen named the [**Sprite Inspector**](#the-sprite-inspector), that allows to see the sprites in action.

> **Disclaimer:** All assets featured in this package have been **generated by AI**. I'm not an artist, I'm an engineer. My contribution in this was to build a toolchain that allows to generate pictures of characters with consistent faces and outfits across different poses (which is not as easy as it sounds). Below is an overview of the workflow I used if you're interested.

<details> 
<summary>AI-generation workflow</summary>
   
1. Generate a reference picture using [Nova Anime XL](https://civitai.com/models/376130/nova-anime-xl) from a description;
2. Build a spritesheet from the reference picture with all desired poses using [Qwen Image Edit](https://huggingface.co/Qwen/Qwen-Image-Edit);
3. Use [Qwen Image Edit](https://huggingface.co/Qwen/Qwen-Image-Edit) again to create variations from the spritesheet (outfits and faces);
    - For faces, use a difference mask between the variation and the base image in order to automatically keep only the areas where the base image was modified;
4. Remove the background in all spritesheets using [isnet-anime](https://huggingface.co/jellybox/isnet-anime/blob/e148f469671b6435c2ad8bb64dfa69507511ebb3/isnet-anime.onnx).
5. Some cleanup may be necessary to remove artifacts in variations and/or portions of the background that weren't automatically removed.

</details>

Download this package and extract it to your project's `images` folder, **or** clone this repository inside your `images` folder.

## The Sprite Inspector

You may invoke the Sprite Inspector by including the following command in your script:

`call screen sprite_inspector`

If you want to add your own character sprites to the Sprite Inspector, you have to do the following:

- Pick a name for your character (example: `alice`)
- Add the name of your character to the `character_sprites.characters` tuple.

Example: `define character_sprites.characters = ("eileen","alice")`

- Add a key with your character's name to the `character_sprites` store, with the following keys inside: `pose`, `outfits`, `faces`. Example:

```rpy
define character_sprites.alice.poses = (
    "front",
    "side",
)
define character_sprites.alice.outfits = (
    "casual",
    "formal",
)
define character_sprites.alice.faces = (
    "neutral",
    "smile",
    "angry",
)
```
